# 💳 EzzyPay-UPI

A modern Android UPI payment application built with **Java** and **Android Studio**, designed to provide a fast, secure, and seamless digital payment experience. EzzyPay enables users to send money, receive payments, check account balance, and manage transactions through a clean and intuitive interface.

---

## 📱 Features

* 🔐 Secure User Authentication
* 💸 Send Money via UPI
* 📥 Receive Payments
* 🏦 Bank Account Integration
* 📊 Transaction History
* 💰 Check Account Balance
* 📱 QR Code Payment Support
* 🔔 Instant Payment Notifications
* 🌙 Clean & User-Friendly UI
* ⚡ Fast and Smooth Performance

## 🚀 Getting Started

### Prerequisites

* Android Studio (Latest Version)
* JDK 11 or above
* Android SDK
* Gradle

### Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/EzzyPay-UPI.git
```

2. Open the project in Android Studio.

3. Sync Gradle files.

4. Run the application on an emulator or physical Android device.
